# Hugo Commands

- Start dev server: ``hugo server -D``
- Compile for deployment: ``hugo -D -d docs/``

# Add Images

1. Delete all images  in ``static/img``
2. Place original files in ``static/img``
3. Run script ``scripts/generate_thumbs.sh``
